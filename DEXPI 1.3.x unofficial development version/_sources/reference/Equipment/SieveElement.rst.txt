.. _dexpi-Equipment-SieveElement-Chamber:

Chamber
=======

.. rubric:: Attribute (reference)


The :doc:`Chamber </reference/Equipment/Chamber>` in which the :doc:`SieveElement </reference/Equipment/SieveElement>` is located, if applicable. The Chamber must be a component of the same object as the :doc:`SieveElement </reference/Equipment/SieveElement>`.

Multiplicity
    0..1

Type
    :doc:`Chamber </reference/Equipment/Chamber>`

Opposite multiplicity
    0..*

.. admonition:: proteus

   The attribute is implemented using :ref:`Proteus \<Association\> elements <proteus-concept-reference-property-association>`.

   Association type for the attribute ower
       ``"is located in"``

   Opposite association type
       ``"is the location of"``

.. admonition:: example

   .. image:: /diagrams/diagram454.*


   .. admonition:: proteus-example
   
      
      .. code-block:: xml
      
        <Equipment
            ID="sieveElement1"
            ComponentClass="SieveElement"
            ComponentClassURI="http://sandbox.dexpi.org/rdl/SieveElement" DOTS123="DOTS123">
          ...
          <Association
            Type="is located in"
            ItemID="chamber1" />
          ...
        <Equipment />
        ...
        <Equipment
            ID="chamber1"
            ComponentClass="Chamber"
            ComponentClassURI="http://data.posccaesar.org/rdl/RDS903151421" DOTS123="DOTS123">
          ...
          <Association
            Type="is the location of"
            ItemID="sieveElement1" />
          ...
        <Equipment />
